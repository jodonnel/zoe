Zoe + Hermes — Quick Start

Your AI assistant running locally on your Linux laptop. 5 minutes to first conversation.

What You're Getting

Zoe

An open-source AI assistant that lives in a Git repo on your machine. She remembers your context across sessions, tracks your work, and helps with account prep, email drafts, meeting notes, research — whatever you throw at her.

Hermes Agent

The terminal interface. Hermes gives Zoe access to your filesystem, a browser, code execution, and a kanban board for tracking work visually. Think of it as the cockpit.

Setup

5 minutes
1

Prerequisites

You need Git, Python 3, and pip. On Fedora/RHEL:

sudo dnf install git python3 python3-pip

Optional but recommended — GitHub CLI for persistent memory:

sudo dnf install gh
gh auth login
2

Run the installer

One command. It installs Hermes, clones Zoe, and walks you through API setup:

bash <(curl -sL https://raw.githubusercontent.com/jodonnel/zoe/main/SCRIPTS/hermes-setup.sh)

Or if you prefer to inspect it first:

curl -sL https://raw.githubusercontent.com/jodonnel/zoe/main/SCRIPTS/hermes-setup.sh -o hermes-setup.sh
cat hermes-setup.sh
bash hermes-setup.sh
3

Choose your model provider

The installer will ask. Easiest options for Red Hat:

Option A: Red Hat MaaS (free, internal)
  • Go to demo.redhat.com
  • Search "Model as a Service"
  • Order the environment — you'll get an API key in the confirmation email
  • Models: Granite 3.2, Qwen3 14B, DeepSeek R1, Llama Scout
Option B: xAI / Grok (free tier)
  • Sign up at console.x.ai
  • Create an API key
  • Model: grok-3 (very capable, fast)
4

Personalize

Open ~/zoe/CLAUDE.md in your editor. Replace the placeholders with your name, your role, and what you're working on. This is how Zoe knows who you are.

5

Launch

cd ~/zoe && hermes

First thing to say: "Sync up. Read my state files and tell me what you see."

The Kanban

This is the thing I wanted you to see. Run hermes board — Zoe breaks your work into tasks, tracks status, and you can see everything at a glance. Say "break this into tasks" during any conversation and she'll populate the board.

Daily Habit

Start every session with "sync up". Zoe reads her state files, knows where you left off, and picks up from there. That's it — one phrase, every time.
